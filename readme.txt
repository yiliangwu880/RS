	RS（Reuse Server Module for C++）
简介：
	C++为主，linux平台一整套可复用服务器组件。以可执行文件，库，工具，代码的方式提供给用户使用。
	欢迎使用，提出问题。
	
设计目的：
	针对游戏服务器，或者类似的服务器。开发完整一套可复用服务器组件。
	消灭重复劳动。快速出稳定的产品。
	使用简单，开发快速，学习成本低。
	支持服务器群，负载均衡，服务器千万级以上用户。
	性能只重视视瓶颈的地方。
	
	服务器开发实践中，都没有高层次的可复用服务器组件。都是在很基础的开源库上自写的。各家不一样，实际很大一部分功能是一样，可以抽象成可复用库。
	网上很多开源优秀服务器组件，自己重造一套可能不是最好的，但是完整一套互相兼容的开源是找不到。 
	
目录结构：
		architectural_model_diagram == 所有组件的依赖关系，用C++类来描述
		com   == 所有组件
