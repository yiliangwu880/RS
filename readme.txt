	Reusable Server for C++
简介：
	C++为主，linux平台一整套可复用服务器组件。
	欢迎免费使用，提出问题。
	
设计目的：
	针对游戏服务器，或者类似的服务器。开发完整一套可复用服务器组件。
	目的：使用简单，开发快速，可复用。 因此放弃 性能最优化。
	
	实践中，遇到的服务器，都没有高层次的可复用服务器组件。都是自撸的。各家不一样，实际很大一部分功能是一样，可以抽象成可复用库。
	网上很多开源优秀服务器组件，自己重造一套可能不比出名的好。
	但是 完整一套互相兼容的找不到。
	
	
目录结构：
